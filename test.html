۱. متغیر چیست؟

متغیر فضایی در حافظه است که می‌تواند مقداری را در خود نگه دارد. در پایتون نیازی به اعلام نوع متغیر پیش از استفاده نیست:

x = 10         # یک متغیر با نام x و مقدار عدد صحیح ۱۰
price = 3.14   # یک متغیر با نام price و مقدار اعشاری
  • پایتون پویاست: نوع متغیر در زمان اجرا تعیین می‌شود.
  • نام متغیر باید با حرف یا آندر‌لاین (_) شروع شود و می‌تواند شامل حروف، اعداد و آندرلاین باشد.
  • از کلمات کلیدی پایتون (مثل for، if، def و…) نمی‌توان به‌عنوان نام متغیر استفاده کرد.

۲. نوع داده int (اعداد صحیح)

نمایانگر اعداد بدون قسمت اعشاری است. می‌تواند مثبت، منفی یا صفر باشد و در پایتون محدودیتی برای طول عدد وجود ندارد.

a = 5
b = -20
c = 0

عملیات روی int

عملیاتمثالنتیجه
جمع3 + 710
تفریق10 - 46
ضرب6 * 742
تقسیم صحیح13 // 52
باقیمانده13 % 53
توان2 ** 38

۳. نوع داده float (اعداد اعشاری)

برای اعداد اعشاری (دارای قسمت کسری) به‌کار می‌رود. تقسیم ساده (/) همیشه خروجی float می‌دهد.

pi = 3.14159
temperature = -5.5
print(7.0 / 2)     # 3.5
print(4 / 2)       # خروجی: 2.0

۴. تبدیل نوع داده (Type Conversion)

گاهی لازم است int را به float یا بالعکس تبدیل کنیم:

x = 10        # int
y = float(x)  # تبدیل به float → 10.0

z = 3.99      # float
w = int(z)    # تبدیل به int → 3 (بخش اعشاری حذف می‌شود)

۵. نکات مهم و بهترین شیوه‌ها

  • نام‌گذاری خوانا: از نام‌های معنادار و snake_case استفاده کنید (مانند total_price).
  • کامنت‌گذاری: برای توضیح کدهای مهم از # استفاده کنید.
  • خوانایی کد: در اطراف عملگرها فاصله بگذارید: result = a + b.
  • اجتناب از بازتعریف: نام‌های داخلی مثل int یا list را بازنویسی نکنید.

۶. مثال جامع

برنامه محاسبه مساحت و محیط دایره:

# ثابت عدد پی
PI = 3.14159

# دریافت ورودی از کاربر
radius_input = input("شعاع دایره را وارد کنید: ")
radius = float(radius_input) # تبدیل رشته به عدد اعشاری

# محاسبه مساحت و محیط
area = PI * radius ** 2
circumference = 2 * PI * radius

# نمایش نتیجه
print("مساحت دایره:", area)
print("محیط دایره:", circumference)

۱. رشته چیست؟

رشته (str) مجموعه‌ای از کاراکترهاست که داخل کوتیشن‌های تکی ('...') یا دوتایی ("...") قرار می‌گیرد:

name = "Ali"
greeting = 'سلام'
  • رشته می‌تواند شامل حروف، اعداد، سمبل‌ها و فضا باشد.
  • برای رشته‌های چندخطی از سه کوتیشن استفاده می‌کنیم:
text = """این یک
رشته‌ی
چندخطی است."""

۲. دسترسی به کاراکترها و طول رشته

ایندکس‌گذاری از صفر شروع می‌شود. طول رشته با تابع len() به‌دست می‌آید:

s = "Python"
print(s[0])   # P
print(len(s)) # 6

۳. برش (Slicing)

می‌توانیم بازه‌ای از کاراکترها را استخراج کنیم ([start:stop]):

s = "Programming"
print(s[0:6])    # "Progra" (از ایندکس ۰ تا ۵)
print(s[3:])     # "gramming" (از ایندکس ۳ تا انتها)
print(s[-3:])    # "ing" (سه کاراکتر آخر)

۴. الحاق و تکرار رشته

با عملگرهای + برای الحاق و * برای تکرار:

print("Hello" + " " + "World")  # "Hello World"
print("ha" * 3)                 # "hahaha"

۵. متدهای رایج str

متدتوضیحمثال
upper()تبدیل همه حروف به حروف بزرگ"hi".upper() → "HI"
lower()تبدیل همه حروف به حروف کوچک"HI".lower() → "hi"
strip()حذف فاصله‌های ابتدا و انتهای رشته" text ".strip() → "text"
split(sep)تقسیم به لیست بر اساس جداکننده"a,b,c".split(",") → ["a","b","c"]
join(iter)الحاق اعضای iterable با رشته"-".join(["a","b","c"]) → "a-b-c"
replace(old, new)جایگزینی زیررشته‌ی old با new"2020".replace("0","1") → "2121"

۶. قالب‌بندی رشته‌ها (Formatting)

۱. با f-string (روش مدرن و پیشنهادی):

name = "Sara"
age = 30
print(f"Name: {name}, Age: {age}")
# خروجی: Name: Sara, Age: 30

۲. با متد .format():

print("Name: {}, Age: {}".format(name, age))
# خروجی: Name: Sara, Age: 30

۷. نکات مهم و بهترین شیوه‌ها

  • خوانایی: برای نمایش متغیرها در خروجی همیشه از f-string استفاده کنید.
  • رشته خام (Raw String): برای جلوگیری از تفسیر کاراکترهای ویژه (مثل \n)، قبل از رشته r بگذارید: path = r"C:\new_folder"
  • رشته‌های طولانی: برای شکستن خطوط طولانی از پرانتز استفاده کنید.

۸. مثال جامع

برنامه‌ای که از کاربر یک جمله می‌گیرد و تعداد کلمات آن را می‌شمارد، همه حروف را بزرگ می‌کند و حروف 'a' را با '@' جایگزین می‌کند:

sentence = input("یک جمله وارد کنید: ")

# شمارش کلمات
words = sentence.split()
count = len(words)

# تبدیل به حروف بزرگ
upper_sentence = sentence.upper()

# جایگزینی
modified = sentence.replace("a", "@").replace("A", "@")

print(f"تعداد کلمات: {count}")
print(f"حروف بزرگ: {upper_sentence}")
print(f"متن ویرایش‌شده: {modified}")

۱. نوع داده بولین (Boolean) چیست؟

bool تنها دو مقدار ممکن دارد:

True     # درست
False    # نادرست
  • حرف اول True و False باید بزرگ باشد (کوچک بنویسید خطا می‌دهد).
  • این نوع داده برای تصمیم‌گیری‌ها (مثل شرط‌ها و حلقه‌ها) کاربرد دارد.

۲. مقایسه‌ها و خروجی bool

عملگرهای مقایسه‌ای مقدار bool بازمی‌گردانند:

عملگرمعنیمثالنتیجه
==مساوی5 == 5True
!=نامساوی3 != 7True
>بزرگ‌تر10 > 5True
<کوچک‌تر2 < 1False
>=بزرگ‌تر یا مساوی5 >= 5True
<=کوچک‌تر یا مساوی4 <= 3False
 
x = 7
print(x > 3)    # True
print(x == 10)  # False

۳. عملگرهای منطقی (and, or, not)

برای ترکیب چند شرط از عملگرهای منطقی استفاده می‌شود:

عملگرتوضیحمثالنتیجه
andاگر هر دو شرط درست باشند(5 > 2) and (4 < 10)True
orاگر حداقل یکی درست باشد(5 > 8) or (2 < 3)True
notمعکوس مقدار بولینnot (5 > 2)False
 
a = 10
b = 5

print(a > 3 and b < 10)   # True
print(a < 3 or b > 10)    # False
print(not a == 10)        # False

۴. کاربرد بولین در شرط‌ها

bool اغلب در if و while استفاده می‌شود:

age = 20
if age >= 18:
    print("بزرگسال هستید")
else:
    print("کم‌سن هستید")

۵. تبدیل انواع دیگر به bool

پایتون به‌صورت خودکار برخی مقادیر را به True یا False تبدیل می‌کند:

مقدارمقدار bool آن
عدد غیر صفرTrue
عدد صفرFalse
رشته غیر خالیTrue
رشته خالی ""False
لیست/دیکشنری غیرخالیTrue
لیست/دیکشنری خالی []/{}False
 
print(bool(5))         # True
print(bool(0))         # False
print(bool("hello"))   # True
print(bool(""))        # False

۶. نکات مهم و شیوه‌های درست

  1. از == True یا == False زیاد استفاده نکنید. به‌جای: 
    if is_ready == True:
    بنویسید: 
    if is_ready:
  2. برای بررسی تهی بودن رشته/لیست: 
    if not my_list:   # اگر خالی است
    print("لیست خالی است")
  3. برای بررسی چند شرط با and/or از پرانتز استفاده کنید تا خوانایی افزایش یابد: 
    if (x > 5 and y < 10):
    ...

۷. مثال جامع

برنامه‌ای که بررسی می‌کند آیا فرد برای دریافت گواهینامه واجد شرایط هست یا نه:

age = int(input("سن خود را وارد کنید: "))
has_id = input("آیا کارت شناسایی دارید؟ (yes/no): ")

# بررسی شرایط
if age >= 18 and has_id.lower() == "yes":
    print("می‌توانید گواهینامه بگیرید.")
else:
    print("شرایط لازم را ندارید.")

۱. لیست (List) چیست؟

لیست یک مجموعه‌ی قابل تغییر (mutable) از مقادیر است که با براکت ([]) تعریف می‌شود:

fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
mixed = ["ali", 25, True, 3.14]
  • لیست می‌تواند انواع داده مختلف را در خود نگه دارد.
  • هر عنصر با , از دیگری جدا می‌شود.

۲. دسترسی به عناصر لیست

با استفاده از ایندکس‌ها می‌توان به عناصر دسترسی داشت:

fruits = ["apple", "banana", "cherry"]
print(fruits[0])    # apple
print(fruits[2])    # cherry
print(fruits[-1])   # cherry (آخرین عنصر)

اگر به ایندکسی که وجود ندارد دسترسی پیدا کنید، خطای IndexError دریافت می‌کنید.

۳. تغییر، افزودن و حذف عناصر

تغییر مقدار:

fruits[1] = "mango"
print(fruits)  # ["apple", "mango", "cherry"]

افزودن:

fruits.append("grape")        # افزودن به انتها
fruits.insert(1, "kiwi")      # افزودن در ایندکس خاص

حذف:

fruits.remove("apple")        # حذف با مقدار
del fruits[0]                 # حذف با ایندکس
popped_item = fruits.pop()    # حذف آخرین و بازگرداندن آن

۴. متدهای رایج لیست

متدکاربردمثال
append(x)افزودن عنصر به انتهای لیستlst.append(10)
insert(i, x)درج عنصر x در ایندکس ilst.insert(1, "ali")
remove(x)حذف اولین وقوع عنصر x از لیستlst.remove("apple")
pop([i])حذف و بازگرداندن عنصر در ایندکس ilst.pop(0)
sort()مرتب‌سازی لیستlst.sort()
reverse()معکوس کردن ترتیب لیستlst.reverse()
count(x)تعداد تکرار عنصر xlst.count(5)
index(x)یافتن اولین ایندکس عنصر xlst.index("apple")
clear()خالی کردن لیستlst.clear()

۵. پیمایش لیست (Loop through a list)

با for:

for fruit in fruits:
    print(fruit)

با ایندکس:

for i in range(len(fruits)):
    print(i, fruits[i])

۶. بررسی وجود یک مقدار در لیست

if "banana" in fruits:
    print("موجود است")
else:
    print("موجود نیست")

۷. لیست‌های تو در تو (Nested Lists)

لیست می‌تواند شامل لیست دیگر باشد:

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matrix[1][2])   # 6

۸. کپی‌کردن لیست

برای جلوگیری از تغییر ناخواسته:

copy1 = original_list.copy()
copy2 = original_list[:]      # روش دیگر

استفاده از = فقط یک ارجاع می‌سازد، نه یک کپی واقعی!

۹. مثال جامع

برنامه‌ای که لیستی از نمرات دانش‌آموزان را گرفته، میانگین آن را محاسبه و وضعیت قبولی را چاپ کند:

grades = []

count = int(input("تعداد نمرات را وارد کنید: "))
for i in range(count):
    g = float(input(f"نمره {i+1}: "))
    grades.append(g)

average = sum(grades) / len(grades)
print(f"میانگین نمرات: {average}")

if average >= 10:
    print("وضعیت: قبول ✅")
else:
    print("وضعیت: مردود ❌")

۱. تاپل (Tuple) چیست؟

تاپل یک مجموعه ترتیب‌دار (ordered) و تغییرناپذیر (immutable) است که با پرانتز () تعریف می‌شود:

coordinates = (4, 5)
colors = ("red", "green", "blue")
  • تاپل می‌تواند شامل هر نوع داده‌ای باشد.
  • پس از ایجاد، نمی‌توان عناصر آن را تغییر داد.

۲. تفاوت لیست و تاپل

ویژگیلیست (list)تاپل (tuple)

۳. ایجاد تاپل

تاپل یک‌عضوی:

باید بعد از مقدار، کاما بگذارید:

single = (5,)     # تاپل با یک مقدار
not_a_tuple = (5) # فقط عدد ۵

مانند لیست‌ها با ایندکس:

t = ("python", 2024, True)

۵. پیمایش تاپل

for item in t:

  • برای داده‌هایی که قرار نیست تغییر کنند

۹. تاپل‌های تو در تو

person = ("Ali", 25, ("Iran", "Tehran"))

۱. دیکشنری (Dictionary) چیست؟

دیکشنری ساختاری نامرتب (unordered) و قابل تغییر (mutable) است که شامل جفت‌های کلید و مقدار (key-value pairs) می‌باشد. با {} تعریف می‌شود.

person = {
    "name": "Ali",
    "age": 30,
    "city": "Tehran"
}
  • کلید (key) باید یکتا و غیرقابل تغییر باشد (مثل str, int, tuple)
  • مقدار (value) می‌تواند هر نوع داده‌ای باشد

۲. دسترسی و تغییر مقدار

دسترسی:

print(person["name"])    # Ali

تغییر مقدار:

person["age"] = 35

افزودن مقدار جدید:

person["job"] = "developer"

اگر کلید وجود نداشته باشد، اضافه می‌شود. اگر وجود داشته باشد، مقدار آن تغییر می‌کند.

۳. حذف عناصر

روشتوضیح
del dict[key]حذف کلید مشخص
dict.pop(key)حذف و بازگرداندن مقدار کلید
dict.clear()خالی‌کردن کامل دیکشنری
 
del person["city"]
job = person.pop("job")
person.clear()

۴. متدهای پرکاربرد دیکشنری

متدتوضیح
keys()لیست کلیدها را برمی‌گرداند
values()لیست مقدارها را برمی‌گرداند
items()لیست جفت‌های (کلید، مقدار)
get(key, default)مقدار کلید را برمی‌گرداند یا مقدار پیش‌فرض
update()به‌روزرسانی با دیکشنری دیگر

۱. مجموعه (Set) چیست؟

set یک ساختار نامرتب (unordered)، بدون عنصر تکراری و قابل تغییر (mutable) است. با آکولاد {} ساخته می‌شود:

my_set = {1, 2, 3, 4}

یا با استفاده از تابع set():

unique_chars = set("hello")   # {'h', 'e', 'l', 'o'}

۲. ویژگی‌های مهم مجموعه‌ها

ویژگیتوضیح
نامرتبترتیب عناصر مشخص نیست (برخلاف لیست)
بدون تکرارفقط یک نسخه از هر مقدار ذخیره می‌شود
قابل تغییرمی‌توان عضو اضافه/حذف کرد
عناصر باید تغییرناپذیر باشندمثل int, float, str, tuple

مثال:

s = {1, 2, 2, 3, 4, 4, 5}
print(s)  # {1, 2, 3, 4, 5}

۳. افزودن و حذف عناصر

افزودن:

s.add(6)             # افزودن یک عنصر
s.update([7, 8, 9])  # افزودن چند عنصر

حذف:

s.remove(3)     # اگر عنصر وجود نداشته باشد خطا می‌دهد
s.discard(10)   # بدون خطا حذف می‌کند اگر وجود نداشته باشد
s.pop()         # حذف تصادفی یک عنصر
s.clear()       # حذف همه عناصر

۴. عملیات ریاضی روی مجموعه‌ها

عملیاتتوضیحمثال
union() یا |اجتماع دو مجموعهA | B
intersection() یا &اشتراک دو مجموعهA & B
difference() یا -تفاضل (فقط در A نه B)A - B
symmetric_difference() یا ^عناصر متفاوت در هر دو مجموعهA ^ B

مثال:

A = {1, 2, 3}
B = {2, 3, 4}

print(A | B)   # {1, 2, 3, 4}
print(A & B)   # {2, 3}
print(A - B)   # {1}
print(A ^ B)   # {1, 4}

if 3 in A:
